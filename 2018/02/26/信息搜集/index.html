google搜索技巧

基本技巧

  1. 与 使用 空格 或 + 将多个关键词隔开（同时含有多个关键词）
  2. 或 使用 OR 将关键词隔开（含有任一关键词）
  3. 非 使用 - 表示不含有某关键词
  4. 完全匹配 使用双引号”关键词”
  5. 通配符 ？ * 分别表示一个字符和一串字符
  6. 指定文件类型 使用filetype：关键词
  7. 指定网站 使用 site：关键词
  8. 与指定网站关联网站 使用related：域名 (查找相似网站)

高级技巧

  1. 页面标题含有关键字 单个关键词使用 intitle：、多个关键词使用 allintitle：
  2. url中含有关键字 使用inurl：、allinurl：
  3. 文本中含有关键字 不包括网页标题和链接，intext：、allintext：
  4. 链接中包含关键字 不包括文本和标题，inlinks：、allinlinks：
